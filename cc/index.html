03 色

  • 03-1 COLOR

  • 03-2 COLOR

  • HSBモードにする

  • 色相・彩度・明度

colorMode(HSB,360,100,100,100);

塗りと線のカラー

図形を描く前に色を設定する

  • 塗りの色はfill()で設定する
fill(300,100,100);
rect(0,0,100,100);
  • 線の色はstroke()で設定する
stroke(200,100,100);
rect(0,0,100,100);

線の太さ

線の太さはstrokeWeight()で設定する

strokeWeight(5);
line(10,200,400,200);

strokeWeight(10);
rect(10,250,100,100);

strokeWeight(15);
ellipse(200,300,100,100);

線を消すには専用の命令がある

noStroke();

背景色を設定できる

background(0,0,100); // white
background(0,0,0); // black

04 変数にする

04 Variable

初めて使う変数には let をつける（宣言）
=を使い、変数に値を代入する

let size;
size = 10;
ellipse(10, 10, size, size); //大きさが10の円を描く

05 繰り返す(for)

05 Repeat

for (let i = 0; i < 10; i++) {
  // 繰り返しのプログラムをここに書く
  // 繰り返しのたびに、変数 i が 0,1,...,8,9 と増えていく
}

06 条件で分ける(if)

06 Condition

もし〜ならば

  if (条件を指定する){
    //条件が成立したとき
  } else {
    //条件が成立しなかったとき
  }

条件は主に比較
==(イコール)や> >=、< <=(不等号)などを使う

07 ランダムにする

07 Random

でたらめな数の表（乱数表）を持っている
範囲を決めて、その間の小数をひとつ取り出す

random(1,7) 1.0 〜 6.999... までの小数

08 マウスの入力

08 Interaction

mouseX と mouseY : マウスカーソル座標を調べるための変数
